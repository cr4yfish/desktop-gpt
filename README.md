# Desktop GPT

Customizable and Privacy Oriented Desktop Assistant.
Run LLMs locally for free, on your own server - or from OpenAI and others.

## Target Features
- [ ] Basic Chat
- [ ] Websearch
- [ ] Screencapture
    - [ ] Manual
    - [ ] Automatic
- [ ] Document Chats
- [ ] RAG on Documents & Screencaptures
- [ ] Call on keyboard shortcut
- [ ] GenUI

## Target LLM support
- [ ] GPT-4o, o1
- [ ] Ollama
- [ ] Gemini
- [ ] Claude models
- [ ] Perplexities models
- [ ] Mistral Models

## Target LLM runtimes
- Local on GPU
- Official Providers (e.g. OpenAI API)
- You own servers

## Stack
Front to back:
- Shadcn UI
- Tailwind
- React
- Langchain
- Typescript
- Deno
- Rust
- Tauri
